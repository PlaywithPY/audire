# Images

Ce dossier contient les images du site Audire.

## Structure

- `logo.png` - Logo principal du site (à uploader)
- `favicon.ico` - Favicon du site (optionnel)
- `uploads/` - Images uploadées via l'interface admin

## Format du logo

**Recommandé :**
- Format : PNG avec fond transparent
- Taille : 200x200px minimum (pour les écrans Retina)
- Poids : < 50 KB

Le logo sera affiché à 34x34px dans le header et footer.
